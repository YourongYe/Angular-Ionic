# class and id
class example:
```css
.question-card{
  // position: absolute;
  height: 200px;
  width: 300px;
  // top: 5%;
  // padding-bottom: 5px;
  visibility: visible;
}
```
id example:
```css
#balloon-effect{ 
  position: absolute;
  width: 100%;
  bottom: -100%;
  transition: ease-in-out;
  animation-name: balloon-animation;
  animation-duration: 5s;
  animation-play-state: paused;
  z-index: 2;
}
```
